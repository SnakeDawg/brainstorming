# Strategy

Product strategy documents: vision, objectives, and the strategic pillars
that roadmap items trace back to. If a roadmap item can't point at a pillar
here, that's a conversation worth having before building it.

- Copy [`_template.md`](_template.md), or

  when it's accepted and closed, the automation files the proposal here.
