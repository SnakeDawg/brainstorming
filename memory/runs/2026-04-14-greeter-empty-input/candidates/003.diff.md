# Candidate 003 — double exclamation for enthusiasm

## Rationale

Candidate 002 was rejected because changing the greeting *word* broke
all tests. Candidate 003 avoids that failure mode: it keeps `"Hello,
Ada"` intact and only adds a second `!` after `Welcome`. The test
assertions are `contains` matches on the `"Hello, X"` substring, so
this change should not regress any test.

The hypothesis is that adding enthusiasm improves the `friendliness`
advisory critic without touching the numeric score. Per the scaffold
rules, advisory critics cannot flip the verdict, so even if
friendliness improves, the loop may not accept without a numeric
delta.

From `lessons.md` after iter 2: "Any change to the greeting prefix
regresses all tests; future candidates must preserve it."
